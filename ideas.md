# ECVN 代理人註冊系統 — 設計方案

## 設計背景
這是一個台灣電力市場（ECVN）的企業內部管理系統，用於代理人註冊與資產綁定。使用者為電力業者的專業人員，需要高效率、清晰的操作體驗。原始 HTML 已有明確的設計語言（深色側邊欄 + 淺色主內容區 + 藍色主色調），我們需要忠實還原。

## 方案一：工業精準主義（Industrial Precision）

**設計運動**：受瑞士國際主義（Swiss International Style）啟發，強調資訊的精準傳達與功能性。

**核心原則**：
1. 資訊密度優先 — 在不犧牲可讀性的前提下，最大化單一畫面的資訊量
2. 功能即裝飾 — 每個視覺元素都必須服務於功能目的
3. 層級分明 — 透過字重、色彩與間距建立清晰的視覺階層

**色彩哲學**：以 Slate 灰階為基底，Blue-600 作為主要行動色（CTA），Emerald-500 表示成功/完成，Red-500 表示警告/刪除。這套配色傳達「專業、可信賴、高效率」的情緒。

**佈局範式**：固定側邊欄 + 固定頂部列 + 可捲動主內容區。三欄式資訊架構。

**標誌性元素**：
1. 步驟進度指示器（圓形數字 + 連接線）
2. 卡片式表單區塊（帶有頂部色帶標識）

**互動哲學**：操作回饋即時且明確 — 按鈕 hover 變色、Modal 帶有背景模糊、表格行 hover 高亮。

**動畫**：極簡過渡 — 側邊欄收合使用 300ms ease-in-out，Modal 使用 opacity + scale 進場，步驟切換使用 fadeIn。

**字型系統**：PingFang TC / Microsoft JhengHei 作為主字型（繁體中文最佳顯示），搭配 monospace 用於編號與代碼欄位。

## 方案二：新擬物暗色系（Dark Neumorphism）

**設計運動**：新擬物主義（Neumorphism）結合暗色模式，營造科技感與深度。

**核心原則**：
1. 觸感模擬 — 透過內外陰影模擬實體按鈕的凹凸感
2. 暗色沉浸 — 深色背景減少視覺疲勞，適合長時間操作
3. 發光引導 — 使用微發光效果引導使用者注意力

**色彩哲學**：以 Slate-900/800 為主背景，Cyan-400 作為主要強調色（帶有微發光），Amber-400 表示警告。暗色系傳達「高科技、前瞻、專業」的氛圍。

**佈局範式**：全暗色沉浸式佈局，卡片使用內凹陰影，按鈕使用外凸陰影。

**標誌性元素**：
1. 發光邊框的進度指示器
2. 帶有內陰影的輸入欄位（模擬凹陷感）

**互動哲學**：按壓回饋 — 按鈕點擊時從外凸變為內凹，營造真實按壓感。

**動畫**：柔和的 glow 脈動效果用於活動狀態，300ms 的陰影過渡。

**字型系統**：Noto Sans TC 搭配 JetBrains Mono，營造科技感。

## 方案三：政府公文風格（Bureaucratic Clarity）

**設計運動**：受日本政府系統設計啟發，極度強調表格化與結構化呈現。

**核心原則**：
1. 表格即介面 — 所有資訊以表格形式呈現，最大化資料密度
2. 色帶分區 — 使用色帶區分不同功能區塊
3. 印章式確認 — 重要操作使用醒目的確認標記

**色彩哲學**：純白底 + 深藍色標題列 + 淺灰色表格交替行。極度克制的用色，僅在狀態標籤使用彩色。

**佈局範式**：垂直堆疊的表格區塊，每個區塊有明確的標題列與邊框。

**標誌性元素**：
1. 帶有編號的區塊標題（如「一、基本資料」）
2. 紅色印章式的完成標記

**互動哲學**：最小化動態效果，強調資料的靜態呈現與列印友善。

**動畫**：幾乎無動畫，僅有基本的 hover 高亮。

**字型系統**：Noto Serif TC 用於標題（正式感），Noto Sans TC 用於內文。

## 最終選擇：方案一 — 工業精準主義

選擇理由：這個方案最忠實於原始 HTML 的設計語言（Slate 側邊欄 + Blue 主色調 + 清晰的卡片式佈局），同時能在 React + TailwindCSS 中完美實現。它專注於功能性與資訊清晰度，完全符合企業內部管理系統的使用情境。
